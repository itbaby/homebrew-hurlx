# Homebrew Tap for hurlx

This is a [Homebrew](https://brew.sh) tap for [hurlx](https://github.com/itbaby/hurlx), an enhanced version of Hurl with import/export support for HTTP testing.

## Installation

```bash
# Add this tap
brew tap itbaby/hurlx

# Install hurlx
brew install hurlx
```

## Upgrade

```bash
brew upgrade hurlx
```

## Uninstall

```bash
brew uninstall hurlx
brew untap itbaby/hurlx
```

## About hurlx

hurlx is an enhanced version of [Hurl](https://hurl.dev), designed for modern API engineering workflows. It fully supports all Hurl features and uniquely adds import/export syntax for modular HTTP testing.

### Key Features

- 🔄 **Modular Import/Export** - Reuse authentication, configuration, and endpoint definitions
- 📊 **Advanced Assertions** - JSONPath, XPath, Regex, type checking, response time
- 🔗 **Variable Chaining** - Chain variables across requests for complex workflows
- 🎯 **Template System** - Dynamic value generation (UUID, dates, random hex, etc.)

Visit [github.com/itbaby/hurlx](https://github.com/itbaby/hurlx) for more details.
